
Какая команда в GIT cmd используется для отправки редактированных файлов в git?
Ответ: Команда git push отправляет изменения в удалённый репозиторий

Что нужно сделать в GIT cmd, чтобы делать изменения от своего имени?
Ответ: Использовать команды git config --global user.name "Ваше имя" и git config --global user.email "Ваш email" для настройки имени и email

Для чего нужен файл id_rsa?
Ответ: Это приватный ключ для аутентификации SSH, используемый для безопасного подключения к удалённым серверам, включая Git
