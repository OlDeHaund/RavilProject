Какая команда в GIT cmd используется для отправки рекадтированных файлов на в git
git add .
git commit -a -m "muudatus"
git push

Что нужно сделать в GIT cmd, чтобы делать изменения от своего имени?
Нужно установить имя и почту пользователя:

git config --global user.name "name"
git config --global user.email "email"

Для чего нужен файл id_rsa?
Это личный ключ для аутентификации по протоколу SSH. Он нужен, чтобы безопасно работать с удаленными репозиториями.
